# 星卷(AstroScroll) 数智文献处理平台 - 规格说明书

## 1. 项目概述

**项目名称**: 星卷(AstroScroll) - 数智文献处理平台
**项目类型**: 单页Web应用 (React + TypeScript + Vite)
**核心功能**: 历史文献浏览、检索、PDF预览、AI对话分析
**目标用户**: 历史研究者、学者、文献爱好者

## 2. 视觉与外观规格

### 整体风格
- **设计语言**: Dark Academia（深色学术风格）融合现代极简主义
- **主色调**: #1a1a2e (深紫黑) / #16213e (深海蓝) / #0f3460 (藏青)
- **强调色**: #e94560 (朱红) / #d4af37 (古金色)
- **辅助色**: #f8f8f2 (象牙白) / #bd93f9 (淡紫)
- **字体**: "Noto Serif SC" (中文宋体), "Inter" (英文)

### 布局结构
```
┌─────────────────────────────────────────────────────────────┐
│  Header: Logo + 导航 + 搜索栏 + 用户头像                      │
├──────────────┬──────────────────────────────────────────────┤
│              │                                              │
│   Sidebar    │              Main Content Area                │
│   导航菜单    │     (文献库 / 详情页 / AI对话 / 分析页)        │
│              │                                              │
│              │                                              │
├──────────────┴──────────────────────────────────────────────┤
│  Footer: 版权信息 + 技术支持                                   │
└─────────────────────────────────────────────────────────────┘
```

### 视觉效果
- 毛玻璃效果 (backdrop-filter: blur)
- 微妙的阴影层次感
- 渐变色块作为视觉引导
- 丝滑的页面过渡动画 (300ms ease)
- 悬浮卡片的光影变化

## 3. 功能模块规格

### 3.1 文献库浏览
- **网格/列表视图切换**: 默认网格视图展示文献卡片
- **筛选功能**: 朝代、类型、作者、关键词
- **排序**: 按时间、热度、相关性
- **文献卡片**: 封面缩略图 + 标题 + 作者 + 朝代 + 标签

### 3.2 检索系统
- **全文搜索**: 支持标题、作者、内容关键词检索
- **智能提示**: 搜索历史 + 热门搜索
- **搜索结果高亮**: 匹配关键词标红

### 3.3 文献详情页
- **PDF预览**: 左侧PDF渲染器，右侧文字稿
- **文字稿**: 可复制、带段落标注
- **元信息**: 作者、朝代、来源、页数、字数统计
- **相关推荐**: 同作者/同朝代/同主题文献

### 3.4 AI对话功能
- **界面**: 侧边抽屉式设计
- **功能**:
  - 基于当前文献的问答
  - 文献内容摘要生成
  - 历史事件时间线梳理
  - 人物关系分析
- **交互**: 打字机效果 + 消息气泡

### 3.5 文献分析功能
- **可视化分析面板**:
  - 词频统计图
  - 情感倾向分析
  - 关键人物提取
  - 事件时间轴
- **分析报告导出**: 生成摘要文档

## 4. 技术栈

- **框架**: React 18 + TypeScript
- **构建**: Vite
- **路由**: React Router v6
- **状态**: Zustand
- **样式**: Tailwind CSS + 自定义CSS变量
- **PDF渲染**: react-pdf
- **图表**: Recharts
- **图标**: Lucide React
- **动画**: Framer Motion

## 5. 页面路由

```
/               - 首页/文献库
/search?q=      - 搜索结果页
/document/:id   - 文献详情页
/analysis/:id   - 文献分析页
/chat           - AI对话页
```

## 6. 组件列表

### 布局组件
- `Layout` - 主布局容器
- `Header` - 顶部导航栏
- `Sidebar` - 侧边导航菜单
- `Footer` - 底部版权栏

### 功能组件
- `DocumentCard` - 文献卡片
- `DocumentGrid` - 文献网格布局
- `SearchBar` - 搜索栏
- `FilterPanel` - 筛选面板
- `PDFViewer` - PDF预览器
- `TextManuscript` - 文字稿组件
- `AIChat` - AI对话界面
- `ChatMessage` - 消息气泡
- `AnalysisPanel` - 分析面板
- `WordCloud` - 词云图

### 状态管理
- `useDocumentStore` - 文献数据状态
- `useChatStore` - AI对话状态
- `useUIStore` - UI状态（侧边栏开关等）

## 8. 验收标准

1. 页面加载时间 < 2秒
2. 搜索响应时间 < 500ms
3. 所有交互有视觉反馈
4. 响应式布局适配桌面和移动端
5. 无控制台错误
6. AI对话流畅，打字机效果正常
7. PDF渲染清晰
8. 动画流畅无卡顿
